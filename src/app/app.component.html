Energy Monitoring System

Easily Monitor And Analyze The Energy Of Various Electrical Devices.

Sensors

Our product enables user to continuously monitor current, voltage, power, energy, frequency, and power factor through the system. In order to collect the real-time data on energy usage, we use PZEM-004T sensors.

IoT

The information received from sensors is sent to IoT server through WiFi using the ESP-32S modules. The data is processed by the specific algorithms and stored in databases. This is handled by MAXIOT - it makes this process much more convenient and efficient.

InfluxDB

We use an InfluxDB that is an open source database and was designed specifically for the time-series data. Influx can cope with a large amount of data and maintain a normal performance at the same time.

Grafana

The processed data should finally be visualized nicely. To do this, we use a web application - Grafana that allows to visualize collected information with charts, diagrams, dashboards, and other appropriate tools so that all information becomes easily perceptible for anyone.
